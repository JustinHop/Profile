Installation

LuaTwitter requires two external modules and, of course, the Lua interpeter installed on your system.

The easiest way (Ubuntu):

  • Install Lua 5.1 and LuaSocket from repositories:
    $ sudo apt-get update
    $ sudo apt-get install lua5.1 liblua5.1-0 liblua5.1-socket2

  • Download the LuaTwitter package, then extract it:
    $ tar -xvzf LuaTwitter.tar.gz

  • Test if everything is well done:
    $ cd LuaTwitter
    $ ./test.lua

After that, you should see some Twitter repsonse. That's all. :)
